# Weather App Frontend

React 19 SPA built with Vite. Part of the [DependencyTrack Weather App Demo](../../docs/20-demo-application/README.md).

## Prerequisites

- Node.js 22.x and npm

## Run locally

```bash
npm install
npm run dev
```

The frontend dev server runs at `http://localhost:5173` by default. It expects the backend at `http://localhost:5063` (configured in `public/app-config.json`).

See [../README.md](../../docs/20-demo-application/README.md) for full setup instructions and an overview of the backend, pipeline, and project structure.
